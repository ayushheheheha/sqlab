# GenzLAB

GenzLAB is an interactive learning platform for students to practice and improve technical skills through hands-on problem solving.

The platform combines a SQL lab, coding challenges, and quiz-based assessment into one unified experience. Students can solve structured problems, run practice queries in an isolated sandbox, attempt Python and Java coding tasks through an online execution engine, and track their growth using progress metrics, streaks, XP, and badges.

GenzLAB is designed for continuous learning with a classroom-friendly workflow: subject-based dashboards, guided hints, instant feedback, leaderboards, and admin-managed content for problems, datasets, users, and quizzes.

At its core, GenzLAB focuses on practical skill-building by making practice consistent, measurable, and engaging.
